# Kupchino News

### Current status
Cleaning and rebuilding project. Migrating to Kotlin.

### Supported versions of Android
- Minimum: Android 4.1.x (API 16)
- Target: Android 7.1 (API 25)<br/>
Note: ~97% of all devices have API 16 and higher
